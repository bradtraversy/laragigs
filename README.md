## Getting Started

To get started
1. run the necesary migrations <br/>
  <code> php artisan migrate </code>
3. Publish the necesary assets <br/>
    <code> php artisan vendor:publish </code>

4. serve the project <br/>
    <code> php artisan serve </code>
